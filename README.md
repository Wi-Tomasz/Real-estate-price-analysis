Analiza i prognozowanie cen mieszkań

Projekt przedstawia kompleksową analizę rynku nieruchomości mieszkaniowych w Warszawie, obejmującą zarówno modelowanie szeregów czasowych, analizę wpływu czynników makroekonomicznych, jak i identyfikację kluczowych determinant cen mieszkań.

Celem projektu jest:
prognozowanie cen mieszkań,
ocena wpływu czynników makroekonomicznych,
identyfikacja kluczowych determinant cen.

Projekt składa się z trzech głównych części:
1. Modelowanie szeregów czasowych (ARIMA)
2. Analiza makroekonomiczna
3. Analiza determinant cen mieszkań

Ograniczenia
brak szczegółowych danych o lokalizacji (dzielnica, ulica),
brak informacji o standardzie wykończenia,
brak danych o stanie technicznym budynku.

Wykorzystane technologie
Python
pandas
numpy
scikit-learn
statsmodels
matplotlib
